¿Qué es CSS y para qué sirve?

CSS es un lenguaje de manejo de diseño y presentación de las páginas web. Funciona junto con el HTML que se encarga del contenido básico de las páginas web. Hace más fácil el trabajo del front-end ya que separa la estructura HTML de la presentación. Sirve para decirle a tu sitio web cómo quieres mostrar la información y guardar los comandos para elementos de estilo.

Formas de aplicar CSS en HTML

1-CSS en línea (inline): se integran en las etiquetas HTML mediante el atributo "style"
2-CSS en la cabecera (internal): se utiliza la etiqueta "style" en la cabecera y se aplicara en todo el código.
3-CSS externo: se consigue al separar el archivo de estilos del fichero HTML, se aplica al fichero HTML mediante el elemento "link". El archivo de estilos cuenta con la extensión .css .
